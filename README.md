# ⚡ ポケモン タイプ相性クイズ

ポケモンのタイプ相性を一問一答形式で学べるクイズアプリです。  
GitHub Pages で無料公開できます。

## 機能

- ⚔️ 攻撃 / 🛡️ 防御の2モードで出題
- 出題するタイプを自由に絞り込み
- 📊 インタラクティブなタイプ相性表（セルホバーでハイライト）
- スマホ・PC 両対応（縦横レスポンシブ）

## ファイル構成

```
pokemon-quiz/
├── index.html   # HTML構造
├── style.css    # スタイル
└── script.js    # データ・ロジック
```

## GitHub Pages で公開する方法

1. このフォルダの中身を GitHub リポジトリにプッシュ
2. リポジトリの **Settings → Pages**
3. Branch: `main` / フォルダ: `/ (root)` を選択して **Save**
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開完了

## ローカルで動かす

ブラウザで `index.html` を直接開くだけで動作します。
